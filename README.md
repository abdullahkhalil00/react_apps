## ⚡ Getting Started

### Prerequisites

• Node.js 18 or later

• npm

### Clone the Repository

```bash
git clone https://github.com/abdullahkhalil00/react_apps.git
cd react_apps
```

### Run Any Project

Navigate to the project directory.

```bash
cd project-folder
npm install
npm run dev
```

Example:

```bash
cd weatherDetail
npm install
npm run dev
```
